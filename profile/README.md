## 弧家军 名称由来
弧家军，起源于一个名叫优弧的人。由于大家经常互相调侃他，并以“弧”为后缀改为自己的昵称，从而形成了一个以“弧”为共同特征的团体。后来，这个团体便被称为“弧家军”。
